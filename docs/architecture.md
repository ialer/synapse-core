## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 应用层 | React/Vue/Swift/Kotlin | 跨平台 UI |
| API 层 | Axum/Actix-web | HTTP 服务 |
| 核心层 | Rust | 高性能核心 |
| 存储层 | SQLite/PostgreSQL/S3 | 数据持久化 |
| 加密层 | ring | AES-256-GCM |
| 异步 | Tokio | 异步运行时 |
| 序列化 | Serde/MessagePack | 数据序列化 |

## 数据模型

### DataEntity

```rust
pub struct DataEntity {
    pub id: Uuid,                    // 唯一标识
    pub owner_id: Uuid,              // 所有者
    pub data_type: DataType,         // 数据类型
    pub encrypted_content: Vec<u8>,  // 加密内容
    pub tags: Vec<String>,           // 标签
    pub created_at: DateTime<Utc>,   // 创建时间
    pub updated_at: DateTime<Utc>,   // 更新时间
    pub version: u64,                // 版本号
    pub is_deleted: bool,            // 软删除标记
}
```

### DataType

```rust
pub enum DataType {
    Credential,  // 凭证
    Config,      // 配置
    File,        // 文件
    Contact,     // 联系人
    Generic,     // 通用
}
```

## 性能优化

1. **异步 I/O**: 所有存储操作使用 Tokio 异步
2. **连接池**: 数据库连接池复用
3. **缓存**: 热点数据内存缓存
4. **压缩**: 大文件透明压缩
5. **批处理**: 批量写入优化
